# 1.项目目标 #
1. 完成二手交易平台的开发
2. 能够完成线上交易
3. 能够展示出发货收货一条完整信息链
4. 保证交易稳定,交易信息可查,交易记录不丢失
5. 保证系统稳定,保护数据不被篡改

# 2.项目范围 #
>### 项目性质
>>### 独立开发
>### 项目组成
>>### 前端,后端,数据库,维护人员
>### 项目要求
>>### 规定时间内完成,能够保证系统稳定运行,数据不丢失,丢失可再找回.
>### 项目范围特殊说明
>>### 无

# 3.项目输入 #  
1. 本产品需要严格遵守国家互联网相关法律法规  
2. 最初可在市级范围内测试，两年内要面向全国开放使用  
3. 本项目按照最少资金选择，同样资金的情况下，用时短优先。

# 4.项目用户 #
任何网络的用户。

# 5.与其他项目的关系 #
除了联合快递公司，快递线路、支付宝银行等之外无特殊关系。

1. 任何项目相关文档、文件属组内机密，不得外泄
